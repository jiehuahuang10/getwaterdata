WPS WebOffice

WebOffice 在线文档预览编辑服务，为接入方提供专业、高质量、易接入的文档处理能力以及相应的增值服务。 支持文字、表格、演示和PDF四大主流文档类型。 支持多人协作编辑，内容实时同步。在安全层面上，WebOffice 分别从传输、存储和代码三个方面采取措施，严格把控文档安全性。
